================================================================================
 TROUBLESHOOTING
================================================================================

Error: ECONNREFUSED 127.0.0.1:3306
  → MySQL not running. Start it: systemctl start mysql

Error: Access denied for user
  → Check DB_USER, DB_PASSWORD in .env
  → Verify user has CREATE DATABASE privilege

Error: Table doesn't exist
  → Import schema: mysql < schema.sql

Error: Cannot find module 'mysql2'
  → Run: npm install

Error: EADDRINUSE :::4173
  → Port already in use. Use different port: PORT=5000 node server.js

Error: JWT verification failed
  → JWT_SECRET changed or token expired (7 days)
  → Login again to get new token
